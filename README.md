# ping-shower


## 编译和安装
首先请安装 `matplotlib`, 本程序使用它作图:
```
sudo apt-get install python-matplotlib python-numpy python2.7-dev
```

然后使用cmake进行进行:
```
cmake .
make
```

## 使用
在Linux上运行需要管理员权限:
```
sudo x86_64/bin/ping_shower 192.168.1.104
```
